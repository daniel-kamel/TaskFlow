## Getting Started

### Prerequisites
- Python 3.9+
- pip (Python package manager)

### Installation
1. **Clone the repository**:
   ```bash
   git clone https://github.com/daniel-kamel/TaskFlow.git
   cd TaskFlow
   pip install -r requirements.txt
   ```

2. **Run the app**:
   ```bash
   python app.py
   ```
